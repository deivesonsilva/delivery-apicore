## Introdução

API para pedidos online (Delivery) de pizza, lanche, bebidas, doces, salgados...

Funcionalidades:

* Cadastrar Comerciantes
* Cadastrar Clientes
* Cadastrar Produtos, Opcionais
* Fluxo de Compra
* Avaliação
* Cashback

## Desenvolvimento

* .Net Core 3.1
* Entity Framework Core - MariaDB
